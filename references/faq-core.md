# Core FAQ

Use these short answers first, then expand with how-to or guide content.

Scope: quick in-product usage questions only (opening Alter, context, settings, dictation, BYOK setup).

## How do I open Alter?

1. Hover over the notch.
2. Press `Alt+Space`.
3. Hold `Fn` and speak.

![Open Alter](https://alterhq.com/assets-doc/images/getting-started/open-notch.gif)

Docs: https://alterhq.com/docs#the-notch

## How do I add my own API key?

1. Open Settings (`Cmd+,`).
2. Go to API Keys.
3. Select provider and enter key.

![Custom API keys](https://alterhq.com/assets-doc/images/models/custom-api-keys.webp)

Docs: https://alterhq.com/docs#external-api-keys

## How do I add context to a prompt?

- Drag and drop files onto the notch.
- Right-click files and choose Add to Alter.
- Use `@` to add context from the prompt.

![Drag and drop](https://alterhq.com/assets-doc/images/context/drag-and-drop.gif)

Docs: https://alterhq.com/docs#the-context-section

## How do I change settings quickly?

- Open Alter and press `Cmd+,`.
- Or open three-dot menu -> Settings.

![Settings menu](https://alterhq.com/assets-doc/images/settings/settings-menu.png)

Docs: https://alterhq.com/docs#accessing-alter-settings

## How do I configure dictation?

Settings -> Dictation, then choose processor, language, and replacements.

![Dictation](https://alterhq.com/assets-doc/images/settings/dictation.webp)

Docs: https://alterhq.com/docs#dictation
